JVM05. Play Framework

Duration : 3 weeks

Prerequisites : FUN05, FUN06

Topics : asynchronous HTTP, REST, web sockets, view templates, persistence

Description : IF not for anything, at least to understand how a web framework should be in today’s times, play is a must learn framework. Stateless web application development paradigm using functional paradigm, with support for asynchronous development similar to nodejs, and excellent concurrency support especially with akka, rails like convention over configuration, sinatra’s simplicity, statically typed with good tooling support in intellij idea makes it an ideal web server framework. With idiomatic scala, without doubt, this is my favourite web application development stack.
