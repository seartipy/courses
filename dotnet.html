.NET courses

NET01 Object oriented programming in C

Duration : 4 weeks

Prerequisites : FUN01

Topics: expressions, statements, collections, classes, events, inheritance, interfaces, generics, exception handling

Description : OOP, generics and error handling could be tough if you are a beginner. I would recommend you to go with FUN01 if you haven’t programmed much. If you already know programming well, this fast paced course will get you upto speed with the most important topics in C#. This course might be the most difficult in .net courses, so be prepared.

NET02 Functional Programming with C

Duration : 4 weeks

Prerequisites : FUN02

Topics: delegates, closures, lambda expressions, extension methods, iterators, type inference, anonymous classes, LINQ to objects

Description : Functional programming allows you to write succint and more maintainable code. C# is a pretty good functional programming language and here you will learn all the features C# supports for functional programming including LINQ, C#’s exhaustive support for comprehensions. If you haven’t done any functional programming before and are a beginner-intermediate programmer, FUN02 is where you should start. Learn F#(NET06), if you need to further your knowledge of functional programming.

NET03 .NET platform

Duration : 3 weeks

Prerequisites : NET01

Topics: garbage collection, reflection, object model, assemblies, MSIL, debugging, performance

Description : This is an advanced course, generally not required unless you work at a lower level, or are curious about the .NET platform. Will be very useful if you are spending lot’s of time perfomance tuning or debugging your .NET applications.

NET04 C# Concurrency

Duration : 4 weeks

Prerequisites : FUN04, NET03

Topics : C# memory model, threads, threadpools, locks, tasks, TPL, PLINQ, async and await

Description : Multithreading is hard. C# provides lower level mechanisms as well as higher level abstractions to write scalable, performant multithreaded code. After learning the lower level details, will mostly spend time on the higher level abstractions provided by .NET/C#. Support for asynchronous development will be discussed. I would highly recommend you learn F#, if you really are interested in learning most patterns used for concurrent code today. C# isn’t enough.

NET05. ASP.NET MVC

Duration : 4 weeks

Prerequisites : NET01, , FUN06

Recommended : NET02, FUN07

Topics : models, validations, views, razor syntax, layout, partials, helpers, controllers, routing, web api, ajax, session handling, unit testing, integration testing, dependency injection and tools.

Description : Inspired by ruby on rails, an excellent example of creating good C# based framework. Heavily uses C# 3.0 and 4.0 features, allowing it to be closer to RoR still remaining true to idiomatic C# usage, and retaining type safety. ASP.NET Web API embracing HTTP, allows you to write modern SPA and/or mobile applications without needing to circumvent many ASP.NET MVC features and you could program in mostly REST style. I highly recommend architecting your web applications based on Web API instead of ASP.NET MVC.

NET06 Functional programming with Fsharp

Duration : 3 weeks

Prerequisites : FUN02

Recommended : NET04

Topics: recursion, functions, immutability, records, higher order functions, pattern matching, workflows, sequences, async, type providers

Description : In today’s world F# is the best programming language on the .NET platform. Excellent support for functional programming and concurrency, one of the best, if not the best support for static typing, decent support for objects(as good as C#), makes it compulsory learning especially if you do not intend to learn haskell, scala or other more famous functional programming languages.
