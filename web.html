Web Development courses

Platform : Mac or linux highly recommended if not mentioned.

WEB01. HTML5

Duration : 2 Weeks

Prerequisites : FUN07

Topics : web storage, web Workers, web sockets, history API

Description : HTML5 made silverlight and flash almost irrelevant. Most features supported by all modern browsers, must if you write mobile web applications. More important parts or the “web” parts of HTML5 are discussed. Almost a requirement if you need to write SPAs today.

WEB02. Asynchronous javascript

Duration : 1 Week

Prerequisites : FUN07

Topics : callbacks, promise/A+, ES6 generators, icedcoffeescript

Description : javascript is asynchronous. This course will explain you what async is, and most modern patterns to deal with callback hell. I do not consider callbacks to be okay.

WEB03. KnockoutJS

Duration : 2 weeks

Platform : Windows recommended

Prerequisites : FUN08

Recommended : WEB03

Topics : bindings, control flow, forms, templates, custom bindings, validation

Description : knockoutjs is one of the simplest, if not the simplest web framework to get started with. It’s a small library, mostly famous on microsoft platform which could be easily integrated with other libraries to create SPAs. DurandalJS a full fledged framework for creating SPAs is built on knockoutjs. Almost all topics will be discussed.

WEB04. AngularJS

Duration : 3 weeks

Prerequisites : FUN08

Topics : controllers, directives, templates, filters, bindings, ajax, dependency injection, routing, events, REST, services, resources, modules, tranclusion, unit testing, integration testing

Description : angularjs is the most famous MV* framework. With google behind it, it has the most chances to succeed compared to other MV* frameworks. It is a complete framework including almost everything needed for creating web applications including unit testing and integration testing. Not a simple framework, but once you get it, you can be very productive.

WEB05. EmberJS

Duration : 3 weeks

Prerequisites : FUN08

Topics : templates, handlebars, routing, components, models, views, enumerables, testing

Description : If you love the “rails way” of web development, you will love emberjs. It’s the most feature rich, complex, opinionated framework. I personally prefer it to almost any other web application framework. It’s clean, elegant and all batteries are included.

WEB06. Mobile Web

Duration : 3 weeks

Prerequisites : FUN07

Topics : jquery mobile, phonegap

WEB07. Coffeescript

Duration : 2 Weeks

Prerequisites : FUN07

Topics : functions, classes, inheritance, functional programming, underscorejs

Description : javascript has good core features, but the syntax is ordinary in my opinion. If you have done ruby or python, I am sure you will understand how painful it could be working with javascript syntax. coffeescript is an excellent choice, if you love the javascript core and ruby/python like syntax. It’s syntax is so good that perhaps you will like it more than ruby/python. coffeescript is just javascript and the code compiles one to one into the equivalent javascript.

WEB08. Typescript

Duration : 2 Weeks

Platform : Windows

Prerequisites : FUN07

Topics : types, functions, classes, interfaces, inheritance, generics, modules

Description : If you love static typing or the tools that come with static typing or you work on windows platform, you might be impressed with typescript. I am mighty impressed with typescript, it has better static typing support compared to most mainstream statically typed languages including C#. Excellent support for generics, interfaces is good enough reason to learn this language even if you aren’t interested in using it.

WEB09. Ruby on rails

Duration : 4 weeks

Prerequisites : FUN06

Topics : activerecord, migrations, validations, associations, queries, views, layouts, partials, helpers, controllers, routing, ajax, asset pipeline, unit testing, integration testing

Description : If you need a complete web framework, productivity is more important than performance, need excellent community support rails is the way to go. I use this framework more often than any other, for great community support, productivity and ruby programming language.

WEB10. NodeJS

Duration : 3 weeks

Prerequisites : FUN07

Recommended : FUN06

Topics : nodejs, socket.io, express

Description : Most talked about stack today. Excellent asynchronous programming support with most if not all API asynchronous allows you to create highly scalable web or network applications especially applications using web sockets heavily. express is a sinatra inspired web framework, with ease of sinatra and scalability of nodejs. socket.io is nodejs implementation of web sockets.

WEB11. Functional Coffeescript

Duration : 2 weeks

Prerequisites : WEB07

Description : javascript, inspired by scheme, is a good functional programming language. Due to it’s dynamic typing, functions as values, closures, allows you to write very good functional code. This course would teach you functional programming techniques inspired by scheme and haskell. As not very useful at work, and perhaps you will learn it only for fun, not including topics. ;-)
